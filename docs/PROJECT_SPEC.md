# CityOps AI — Project Specification

## Overview

CityOps AI is an AI-powered city operations management platform that enables citizens to report urban issues and city operations teams to manage, prioritize, and resolve them using AI-assisted workflows.

## Architecture

- **Backend**: Node.js + Express + TypeScript
- **Frontend**: React + Vite + TypeScript
- **Cloud**: Firebase (Firestore, Cloud Storage), Google Cloud Run
- **AI**: Google Gemini (via Gen AI SDK)
- **Maps**: Google Maps Platform
